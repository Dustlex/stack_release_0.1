1) .env с доступами тут для наглядности, на хосте к которому будет применяться плейбук и где будет развернута инфраструктура никаких доступов не будет ни в файлах ни в репах кроме окружения внутри контейнеров.

2) запуск простой командой ansible-playbook -i hosts.yml all.yml , предварительно нужно убедиться что в hosts.yml прописан нужный IP сервера к которому нужно применить плейбук, и то что ключ к root для него прокинут

3) Есть два тега по одному на роль. Их можно использовать если плейбук целиком не нужен.

Только установить докер:
```
ansible-playbook -i hosts.yml all.yml --tags stack
```
Только развернуть контейнеры если докер уже установлен:
```
ansible-playbook -i hosts.yml all.yml --tags deploy
```
