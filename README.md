.env с доступами тут для наглядности, на хосте к которому будет применяться плейбук и где будет развернута инфраструктура никаких доступов не будет ни в файлах ни в репах кроме окружения внутри контейнеров.
запуск простой командой ansible-playbook -i hosts.yml all.yml , предварительно нужно убедиться что в hosts.yml прописан нужный IP сервера к которому нужно применить плейбук, и то что ключ к root для него прокинут
